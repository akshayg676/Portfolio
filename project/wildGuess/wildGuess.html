    • WILD GUESS

    Description:

    Wild Guess is a simple Image Classification / Identification app using React and Tensorflow JS. Tensorflow JS is a machine learning library for Javascript.
    In this project I use the pre-trained machine learning model from Tensorflow JS called mobilenet. This model is specifically used for image classification. This is a very simple app that uses tensorflow model, and tries to identify the content of the image. So if we use an image like animals, birds, fruits or other simple things, The Tensorflow will work behind the scenes to identify the content for this image, and will display the results in 3 guesses.
